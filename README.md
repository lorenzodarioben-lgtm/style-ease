# Style Ease

A small static Vue 2 fashion storefront.

## Run Locally

Open `src/index.html` in a browser, or serve the folder with any static file server.

The app uses Vue and Vue Router from CDN links, so an internet connection is needed unless those scripts are vendored locally.

## Structure

- `src/index.html` loads the app shell and scripts.
- `src/css/` contains the stylesheet sections in cascade order.
- `src/js/data/` contains catalogue and site content.
- `src/js/utils/` contains shared helper functions.
- `src/js/components/` contains reusable Vue components.
- `src/js/pages/` contains route-level Vue pages.
- `src/js/router.js` defines routes.
- `src/js/app.js` boots the Vue app.
